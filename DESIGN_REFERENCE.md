# Design Reference Instructions

The screenshots inside this project are the visual source of truth.

Reference Files:

- desktop-reference.png
- mobile-reference.png

Requirements:

- Match layout exactly
- Match spacing exactly
- Match typography closely
- Match colors closely
- Match gradients closely
- Match card sizes closely
- Match border radius closely
- Match shadows closely
- Match visual hierarchy closely

Workflow:

1. Analyze reference screenshots.
2. Build UI.
3. Capture screenshots.
4. Compare against references.
5. Identify differences.
6. Fix differences.
7. Compare again.
8. Repeat until visually very close.

Do not redesign.

Do not simplify.

Do not add extra sections.

The goal is recreation, not reinterpretation.

Desktop screenshot controls desktop layout.

Mobile screenshot controls mobile layout.

Both must be implemented.
